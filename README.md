# product-category-classifier
# Product Category Classifier

## Опис пројекта
Овај пројекат представља систем за **аутоматску класификацију производа** по категоријама на основу њихових назива.  
Пројекат је развијен као пример примене **машинског учења** у online трговини, где велики број производа мора бити брзо и прецизно категорисан.

Модел омогућава:
- Аутоматско предвиђање категорије новог производа.
- Бржу обраду и унос производа.
- Смањење могућности људске грешке.
